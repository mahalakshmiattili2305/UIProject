# Bootstrap 5 UI Exploration Project

## Project Overview
This project was created as part of an internship task to explore Bootstrap 5 and design clean, responsive web pages by remixing Bootstrap components.

## Step-by-Step Process
1. Studied Bootstrap 5 documentation and example pages.
2. Identified reusable UI components such as navbars, cards, forms, and grids.
3. Designed three pages: Home, About, and Contact.
4. Combined components creatively instead of copying full examples.
5. Tested responsiveness on different screen sizes.

## Tools & Resources Used
- Bootstrap 5 official documentation
- Bootstrap Examples
- VS Code
- GitHub'
- Stackoverflow
- Used bootstrap CDN from bootstrap documention

## Challenges Faced
- Understanding Bootstrap grid layout
- Maintaining consistent spacing and alignment
- Making layouts responsive
- selecting colors

## How I Solved Them
- Referred to Bootstrap documentation
- Used Bootstrap utility classes
- Tested layouts by resizing the browser
- by testing with different colors I finally which looks better for official web page

- Gained confidence in building structured and simple web pages
